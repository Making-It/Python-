# Python语法

## 内置函数

### zip函数

[菜鸟教程](<https://www.runoob.com/python3/python3-func-zip.html>)

* 用于将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的对象，这样做的好处是节约了不少的内存

* 可以使用 list() 转换来输出列表
* *符号可以将列表拆分成组元

```python
a = [1,2,3]
b = [4,5,6]
c = [7,8,9]

z = list(zip(a,b,c))
print(z)
#打印 [(1, 4, 7), (2, 5, 8), (3, 6, 9)]

print(*z)
#打印 (1, 4, 7) (2, 5, 8) (3, 6, 9)

print(list(zip(*z)))
#打印 [(1, 2, 3), (4, 5, 6), (7, 8, 9)]
```
